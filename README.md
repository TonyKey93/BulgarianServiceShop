# Сервиз за ремонт — Дигитален бележник

Локално web приложение за приемане и проследяване на ремонти на телевизори и домашни уреди. Проектирано за лесна употреба — големи букви, висок контраст, минимален интерфейс.

## Технологии

- **Next.js 15** (App Router)
- **React 19**
- **Tailwind CSS 4**
- **SQLite** + **Prisma ORM** (работи напълно офлайн на локален компютър)

## Изисквания

- [Node.js](https://nodejs.org/) версия **18.18+** или **20+** (препоръчително LTS)
- npm (идва заедно с Node.js)

## Инсталация и стартиране

Отворете терминал в папката на проекта (`serviz-remonti`) и изпълнете:

```bash
# 1. Инсталиране на зависимостите
npm install

# 2. Създаване на SQLite базата данни
npm run db:push

# 3. (По избор) Примерни данни за тест
npm run db:seed

# 4. Стартиране на приложението
npm run dev
```

Отворете браузъра на адрес: **http://localhost:3000**

### Ако `npm install` дава SSL грешка (Windows)

Някои системи изискват системните сертификати:

```powershell
$env:NODE_OPTIONS="--use-system-ca"
npm install
```

## Полезни команди

| Команда | Описание |
|---------|----------|
| `npm run dev` | Стартира приложението в режим разработка |
| `npm run build` | Компилира за production |
| `npm run start` | Стартира production сървъра (след `build`) |
| `npm run db:push` | Синхронизира схемата с SQLite базата |
| `npm run db:seed` | Добавя примерни ремонти |
| `npm run db:studio` | Отваря Prisma Studio (визуален преглед на базата) |

## Функционалности

- **Табло (Dashboard)** — един екран с таблица като дигитален бележник
- **Нов ремонт** — голям бутон за добавяне на клиент/уред
- **Търсене** — по име или телефон
- **Филтър** — по статус (Приет, Чака части, Готов, Издаден)
- **Статуси** — цветни етикети, кликваеми директно в таблицата

## Статуси

| Статус | Цвят |
|--------|------|
| Приет | Сиво/синьо |
| Чака части | Оранжево/жълто |
| Готов | Зелено |
| Издаден | Тъмно/приглушено |

## База данни

Файлът на базата се съхранява локално в:

```
prisma/dev.db
```

Данните остават на вашия компютър — не се изисква интернет след инсталация.

## Бележки за употреба

1. Натиснете **„+ Нов ремонт“** за добавяне на запис.
2. Кликнете върху **статуса** в таблицата, за да го смените без да отваряте нов екран.
3. Използвайте **търсенето** за бързо намиране по телефон или име.
4. Телефонните номера са кликваеми — отварят директно обаждане на мобилен телефон.
